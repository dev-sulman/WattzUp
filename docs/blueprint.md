# **App Name**: WattzUp

## Core Features:

- Service Type Selection: Allow users to select the type of electrician service they need (Residential, Commercial, Industrial, Specialized, Emergency).
- Detailed Service Information: Provide detailed information about each service type, including descriptions, sub-services offered (e.g., wiring, lighting, repairs), and approximate pricing.
- Electrician Finder: Search for electricians based on location and service type.  Incorporate electrician ratings and reviews.
- Cost Estimator: Provide an estimated cost calculator tool that uses details from the prompt to give price range depending on selected service.
- Emergency Support: Feature to instantly call electricians during power outage, electrical sparks and/or burning smells

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) to convey professionalism and energy.
- Background color: Light gray (#E0E0E0) to provide a neutral backdrop.
- Accent color: Yellow-orange (#FFB347) to highlight key interactive elements.
- Body and headline font: 'Inter' for a clean and modern appearance.
- Use clear and simple icons representing electrical services, tools, and safety elements.
- Clean and structured layout, using cards to organize services and information.
- Subtle animations when displaying new service information, and on user interactions with the search function.
